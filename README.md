# ReactJS DEMO

## 用到技术栈 react + react-router / ES6 + Babel / webpack + webpak-dev-server / ant / mockJS

## 功能

	1、登录、登退

	2、路由、子路由

	3、路由跳转调钩子来判断是否登录

	4、ant 来处理ui界面

	5、Mock数据

## 安装使用

	git clone

	npm install

	cnpm i open-browser-webpack-plugin  html-webpack-plugin --save		// webpack插件

	npm run dev		// 生产环境开发

	webpack 		// 打包到build目录用于线上环境使用
